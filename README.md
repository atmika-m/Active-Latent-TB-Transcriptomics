# # Comparative Transcriptomic Profiling of Active and Latent Tuberculosis

## Overview

This project investigates host-gene expression signatures that distinguish **Active Tuberculosis (ATB)** from **Latent Tuberculosis Infection (LTBI)** using publicly available whole-blood microarray datasets.

The study integrates transcriptomics, bioinformatics, and machine learning to identify candidate biomarkers associated with tuberculosis progression.

---

## Objectives

- Identify differentially expressed genes (DEGs)
- Perform GO and KEGG enrichment analysis
- Prioritize biomarkers using Random Forest
- Construct Protein–Protein Interaction (PPI) networks
- Identify hub genes using Cytoscape
- Evaluate hub genes using survival analysis

---
