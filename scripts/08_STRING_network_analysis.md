# Protein–Protein Interaction Network Analysis

## Tool
STRING Database (https://string-db.org)

## Input

Top 50 genes prioritized by the Random Forest model.

## Parameters

- Organism: Homo sapiens
- Minimum interaction score: 0.900 (Highest confidence)
- Disconnected nodes: Hidden

## Output

- Protein–Protein Interaction (PPI) Network
- Network exported for Cytoscape visualization.
