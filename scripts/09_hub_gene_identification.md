# Hub Gene Identification

## Software

Cytoscape

## Plugin

cytoHubba

## Ranking Algorithm

Maximal Clique Centrality (MCC)

## Procedure

- Import STRING network.
- Run cytoHubba.
- Rank genes using MCC.
- Export Top 25 hub genes.

The highest-ranked genes included:

- B2M
- TYROBP
- FCGR3A
- HLA-E
- LILRB3

These genes were selected for downstream survival analysis.
